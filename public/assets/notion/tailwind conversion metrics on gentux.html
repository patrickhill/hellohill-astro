Initial notes


March 3, 2021 - Sept 14, 2021

(195 days, ~28 weeks, ~6+ months)


What was done


  • removed component library
  • removed styled components
  • converted project to typescript
  • move theming into tailwind configuration (rather than react+styledcomponents)
  • wrote ~80 new components using typescript + tailwind + jsx

Why


Reduce the systemic bloat of component library across projects. Decouple front-end repos from one another and their tech stack.


Side effects


  • allows for project to be moved into docker
  • much cleaner codebase (easier for dev and maintenance)
  • most components use standard HMTL tags again
  • documentation for free at tailwindcss.com
  • integrated new accessible form components
  • home page line difference on page load (7760lines vs 1047lines)
  • daily build time ~30-45 seconds faster
